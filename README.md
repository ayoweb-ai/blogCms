Blog CMS is a simple static content management system built with HTML, CSS, and JavaScript.

It lets users add, edit, and delete blog posts all stored locally using localStorage (no backend required).

Features

-Clean, responsive landing page with hero, features, and contact section

-Simple dashboard to create, edit, and delete posts

-LocalStorage based blog management (data persists locally)

-Static design for smooth navigation between pages

-Minimal UI and easy-to-use interface

Live link: https://ayoweb-ai.github.io/blogCms/
